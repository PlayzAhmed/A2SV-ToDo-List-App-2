# Setup

1. Clone this repository (or copy the files into a new folder):
   ```bash
     git clone [https://github.com/PlayzAhmed/A2SV-ToDo-List-App-2.git](https://github.com/PlayzAhmed/A2SV-ToDo-List-App-2.git)
     cd A2SV-ToDo-List-App-2
   ```
2. Running the app:
   ```bash
     node ./dist/index.js
   ```
